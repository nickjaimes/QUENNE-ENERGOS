Local Development Setup

```bash
# Clone the repository
git clone https://github.com/quenne-energos/quenne-core.git
cd quenne-core

# Install dependencies
make install

# Set up environment
cp .env.example .env
# Edit .env with your configuration

# Start development environment
make dev

# Run tests
make test

# Start simulation environment
make simulate-grid
```

Docker Deployment

```bash
# Build and run the core stack
docker-compose up -d

# Scale services
docker-compose up -d --scale quantum-worker=4 --scale neuromorphic-node=8

# Monitor system
docker-compose logs -f
```
